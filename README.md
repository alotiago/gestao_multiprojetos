## 📖 Sobre o Projeto

O **Gestão de Multiprojetos** é uma plataforma para centralizar e acompanhar o andamento de múltiplos projetos de forma simultânea. O sistema permite que gestores e equipes tenham uma visão unificada do portfólio de projetos, facilitando a alocação de recursos, o monitoramento de prazos e a comunicação entre times.

### Objetivos principais

- Centralizar informações de todos os projetos em um único local.
- Oferecer dashboards com indicadores de desempenho (KPIs) por projeto e por portfólio.
- Facilitar a gestão de recursos humanos e orçamentários entre projetos.
- Registrar histórico de decisões, riscos e entregas de cada projeto.

## ✨ Funcionalidades

| Funcionalidade | Descrição |
|---|---|
| **Cadastro de Projetos** | Criação e edição de projetos com informações detalhadas |
| **Gestão de Tarefas** | Criação, atribuição e acompanhamento de tarefas por projeto |
| **Controle de Equipes** | Gerenciamento de membros, papéis e permissões |
| **Gestão de Recursos** | Alocação e monitoramento de recursos humanos e materiais |
| **Dashboard** | Visão consolidada do portfólio de projetos com indicadores |
| **Relatórios** | Geração de relatórios de progresso, status e desempenho |
| **Cronograma** | Visualização de cronogramas e interdependências entre projetos |
| **Gestão de Riscos** | Registro e acompanhamento de riscos e planos de mitigação |
| **Notificações** | Alertas automáticos sobre prazos e mudanças de status |
