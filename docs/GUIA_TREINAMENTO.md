# Guia de Treinamento — Gestor Multiprojetos

**Público-alvo:** Usuários-chave e Administradores  
**Duração estimada:** 4 horas (2 sessões de 2h)

---

## Sessão 1 — Usuários-Chave (2h)

### Módulo 1: Introdução (20 min)

**Objetivo:** Familiarizar os participantes com o sistema e a interface.

1. **O que é o Gestor Multiprojetos**
   - Substituição da planilha PR_SEEC_2026
   - Módulos: Projetos, RH, Financeiro, Operações, Configurações
   - Vantagens: controle de acesso, auditoria, cálculos automáticos

2. **Acesso ao sistema**
   - URL de acesso
   - Login com email e senha
   - Perfis e permissões (o que cada um pode fazer)

3. **Interface principal**
   - Sidebar (navegação)
   - Dashboard (visão geral)
   - Padrão de tabelas (busca, filtros, criar, editar)

### Módulo 2: Módulo Projetos (30 min)

**Objetivo:** Criar, editar e buscar projetos.

**Exercícios práticos:**
1. Criar um novo projeto de teste
2. Editar o status de um projeto existente
3. Buscar um projeto pelo código
4. Visualizar detalhes de um projeto

### Módulo 3: Módulo RH (30 min)

**Objetivo:** Gerenciar colaboradores e jornadas.

**Exercícios práticos:**
1. Cadastrar um novo colaborador
2. Registrar jornada mensal
3. Fazer upload de um arquivo CSV de teste
4. Buscar colaborador por nome

### Módulo 4: Módulo Financeiro (20 min)

**Objetivo:** Consultar dados financeiros.

**Exercícios práticos:**
1. Visualizar despesas por projeto
2. Consultar provisões
3. Verificar índices financeiros

### Módulo 5: Dashboard e Exportação (20 min)

**Objetivo:** Usar o Dashboard para visão gerencial.

**Exercícios práticos:**
1. Interpretar os KPIs do Dashboard
2. Verificar alertas do sistema
3. Exportar dados para CSV

---

## Sessão 2 — Administradores (2h)

### Módulo 6: Configurações (30 min)

**Objetivo:** Configurar calendários e sindicatos.

**Exercícios práticos:**
1. Criar calendário para o ano atual
2. Executar seed de feriados
3. Cadastrar um sindicato
4. Simular impacto de dissídio

### Módulo 7: Operações em Massa (30 min)

**Objetivo:** Executar recálculos e operações em lote.

**Exercícios práticos:**
1. Executar recálculo cascata para um projeto
2. Atualizar jornadas em massa
3. Verificar histórico de operações

### Módulo 8: Administração do Sistema (30 min)

**Objetivo:** Gerenciar usuários e perfis.

**Tópicos:**
1. Criar novo usuário
2. Definir role/perfil de acesso
3. Desativar usuário
4. Consultar logs de auditoria

### Módulo 9: Monitoramento e Backup (30 min)

**Objetivo:** Garantir a saúde operacional do sistema.

**Tópicos:**
1. Verificar health check do sistema
2. Acessar Grafana para métricas
3. Procedimento de backup do banco
4. O que fazer em caso de problemas (troubleshooting)
